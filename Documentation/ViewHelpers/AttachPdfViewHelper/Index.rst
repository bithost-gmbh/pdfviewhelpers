AttachPdfViewHelper
-------------------

This ViewHelper allows to import all pages of an existing PDF document.

::

	<pdf:document>
		<pdf:attachPdf path="EXT:pdfviewhelpers/Resources/Public/Examples/ExtendExistingPDFs/pdf_template.pdf" />
	</pdf:document>
